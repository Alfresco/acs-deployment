You can access all components of Alfresco Content Services using the same root address, but different paths as follows:

  Content: {{ .Values.externalProtocol }}://{{ .Values.externalHost }}:{{ .Values.externalPort }}/alfresco
  Share: {{ .Values.externalProtocol }}://{{ .Values.externalHost }}:{{ .Values.externalPort }}/share
  Solr: {{ .Values.externalProtocol }}://{{ .Values.externalHost }}:{{ .Values.externalPort }}/solr

{{ else }}

If you have a specific DNS address for the cluster please run the following commands to get the application paths:

helm upgrade --reuse-values {{ .Release.Name }} --set externalProtocol="http" --set externalHost="domain.com" --set externalPort="80" alfresco-incubator/alfresco-content-services

{{ end }}
